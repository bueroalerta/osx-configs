* /usr/bin/ruby -e "$(/usr/bin/curl -fsSL https://raw.github.com/mxcl/homebrew/master/Library/Contributions/install_homebrew.rb)"
* RVM (https://rvm.io/rvm/install/)
* curl -L https://get.rvm.io | bash -s stable
